# feedwatch

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run feedwatch add <url>
```

```bash
bun run feedwatch list
```
```bash
bun run feedwatch remove <url>
```
```bash
bun run feedwatch run
```
```bash
bun run feedwatch run all
```
```bash
bun run feedwatch run json
```
```bash
bun run feedwatch read <url>
```

This project was created using `bun init` in bun v1.3.10. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
